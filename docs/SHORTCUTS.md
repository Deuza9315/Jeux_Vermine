# Raccourcis Clavier

## Gameplay

| Touche | Action |
|--------|--------|
| ↑ Haut | Déplacer vers le haut |
| ↓ Bas | Déplacer vers le bas |
| ← Gauche | Déplacer vers la gauche |
| → Droite | Déplacer vers la droite |
| Z | Haut (AZERTY) |
| Q | Gauche (AZERTY) |
| S | Bas (AZERTY) |
| D | Droite (AZERTY) |
| Entrée | Recommencer (Game Over) |

## Conseils

### Positionnement
- Reste au centre pour avoir plus d'espace
- Les coins sont des pièges
- Anticipe la trajectoire des ennemis

### Gestion des Bonus
- Priorise +Vie quand tu as 1-2 vies
- Utilise Ralenti pour t'échapper
- Garde Bouclier pour les moments critiques
- Déclenche Bombe quand plusieurs ennemis convergent

### Scoring Optimal
- Chaque seconde compte (x10)
- Les bonus ne donnent pas de points directs
- Prolonger la survie = meilleur score
- 100 secondes = 1000 points

## Combos Gagnants

### "Slow-Mo Escape"
1. Attendre Ralenti + Bouclier
2. Bouger au centre
3. Attendre que tout passe
4. Ralenti crée du temps supplémentaire

### "Bomb Survival"
1. Accumuler ennemis au même endroit
2. Déclencher Bombe
3. Spawner de nouveaux ennemis = nouvelle vie possible

### "Life Extension"
1. Chercher +Vie activement
2. Garder maximum de vies (5)
3. Plus de marge d'erreur = plus longtemps en jeu
