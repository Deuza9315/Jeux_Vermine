# Changelog

Tous les changements majeurs de ce projet sont documentés dans ce fichier.

## [1.0.0] - 2026-05-05

### Ajouté
- 🦇 Chauve-souris CSS pure avec animation 7 frames
- 🐍 Serpent à sonnette avec animation tête/corps
- 🕷️ Araignée venimeuse avec 16 sprites animés
- ⚡ Système de 4 bonus (vie, ralenti, bouclier, bombe)
- 🔊 Sons générés par Web Audio API
- 📊 Leaderboard top 5 avec localStorage
- 🎮 Contrôles AZERTY et QWERTY
- 📱 Interface responsive

### Gameplay
- Difficulté progressive avec le temps
- Système de vies + invincibilité temporaire
- Collision AABB optimisée
- Spawn aléatoire de bonus
- Scoring par secondes survécues

### Technique
- HTML5/CSS3/JS vanilla
- Zero dépendances
- Un seul fichier (230 Ko)
- Assets embarqués en Base64
- Jouable hors ligne
