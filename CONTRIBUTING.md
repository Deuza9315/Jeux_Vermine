# Guide de Contribution

Merci de ton intérêt pour le projet VERMINE ! 🦇

## Comment Contribuer

### Signaler un Bug

Si tu trouves un bug:
1. Vérifie qu'il n'a pas déjà été signalé
2. Décris le comportement attendu vs réel
3. Donne les étapes pour le reproduire
4. Inclus des screenshots si possible

### Suggérer une Amélioration

Les idées sont toujours bienvenues!
- Ennemis additionnels
- Nouveaux bonus
- Modes de jeu
- Améliorations visuelles/sonores

## Standards de Développement

### Code
- Vanilla JS (pas de frameworks)
- Noms de variables explicites en français/anglais
- Commentaires pour la logique complexe
- Keep it simple

### Commits
- Messages clairs et concis
- Prefix type: feat:, fix:, docs:, style:, refactor:
- Français ou anglais

### Testing
- Test le gameplay
- Vérifiez les collisions
- Test sur plusieurs navigateurs
- Vérifiez la performance

## Processus

1. Fork le repo
2. Crée une branche (`feature/ma-feature`)
3. Commit tes changements
4. Push vers ton fork
5. Ouvre une Pull Request

## Questions ?

N'hésite pas à ouvrir une issue ou une discussion !

Merci pour ton aide! 🎮
